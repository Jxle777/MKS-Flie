# 马克思哲学入门研读资料

一个现代化的马克思主义哲学学习资料浏览系统，提供系统化的学习资料和便捷的文件管理功能。

## 项目特色

- 🎨 **现代化设计** - 采用红色主题，符合马克思主义特色
- 📱 **响应式布局** - 支持各种设备访问
- 📂 **文件浏览** - 直观的文件管理系统
- 🔍 **分类浏览** - 按主题分类的学习资料
- 📖 **在线预览** - 支持文本文件在线预览
- ⚡ **纯前端** - 无需服务器，直接打开即可使用

## 配置与使用说明

### 动态文件配置（config.js）
- 所有文件与目录通过 `window.dynamicFileConfig` 进行配置，键为目录路径（如 `/`、`/马主义哲学入门研读资料`）。
- 文件对象字段：
  - `name`：文件/文件夹名称
  - `type`：`'file' | 'folder'`
  - `size`：文件大小，文件夹为 `'-'`
  - `modified`：修改时间，建议格式 `YYYY-MM-DD HH:mm`
  - `path`：以 `/` 开头的完整路径（与 `download/` 下真实路径对应）

示例：
```javascript
// 在控制台动态添加一个文件
addNewFile(
  '/马主义哲学入门研读资料',
  '新阅读材料.pdf',
  'file',
  '2.5 MB',
  '/马主义哲学入门研读资料/新阅读材料.pdf'
);
```

常见注意事项：
- 路径需与 `download/` 实际文件一致，区分大小写与全角字符
- 文件必须真实存在，否则会显示为空
- 目录若要展示内容，必须在配置中存在对应的键（例如：`'/…/马克思主义哲学史': [ ... ]`）

### 文件交互功能
- 文本（txt, md, json, js, css, html, xml）：在弹窗内直接预览、支持下载与新窗口打开
- 图片（jpg, jpeg, png, gif, bmp）：弹窗预览、下载、新窗口打开
- PDF：支持下载与新窗口打开
- 其他文档（docx, xlsx, pptx 等）：下载与新窗口打开

提示：
- 在 HTTP/HTTPS 下可直接读取文本以展示内容；在 `file://` 本地直接打开时，浏览器限制可能无法读取，建议使用“在新窗口阅读”或“下载文件”。

### 子目录映射指引
- 若某子目录显示为空，通常是因为缺少对应的配置键。请在 `config.js` 中补充：
```javascript
window.dynamicFileConfig['/父目录/子目录'] = [
  { name: '文件A.pdf', type: 'file', size: '10 MB', modified: '2024-07-14 12:56', path: '/父目录/子目录/文件A.pdf' },
  // ...更多文件
];
```
- 示例：`'/马主义哲学入门研读资料/马克思主义理论 入门必读(先读这里)/马克思主义哲学史'` 已配置了完整条目，可参考其格式为其他目录补充。
